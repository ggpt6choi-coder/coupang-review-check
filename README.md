# 🚀 쿠팡 리뷰 크롤러 (Coupang Review Crawler)

이 프로젝트는 쿠팡의 특정 카테고리 내 상품들을 분석하여 최신 리뷰가 있는 유망 상품을 자동으로 수집하는 도구입니다. 웹 대시보드를 통해 실시간으로 진행 상황을 확인하고 수집된 데이터를 관리할 수 있습니다.

## ✨ 주요 기능
- **웹 대시보드 UI**: 실시간 로그 확인, 통계 모니터링, 크롤링 설정 제어.
- **스마트 필터링**: 최소/최대 가격 범위 설정, 리뷰 작성 기간 필터링(예: 최근 30일 이내).
- **자동 저장 및 로드**: 설정을 자동으로 `localStorage`에 저장하여 재접속 시 유지.
- **이미 처리된 상품 건너뛰기**: `result.json`을 기반으로 중복 수집을 원천 차단.
- **데이터 관리**: 수집된 데이터를 JSON으로 즉시 다운로드하거나 초기화 가능.
- **안전한 서비스 종료**: 크롬 브라우저와 서버를 한 번에 종료하는 시스템 종료 버튼.

## 🛠 설치 방법 (새로운 PC 기준)

1.  **파일 복사**: 이 저장소의 모든 파일을 수집을 원하는 PC로 가져옵니다.
2.  **Node.js 설치**: [Node.js 공식 홈페이지](https://nodejs.org/)에서 최신 LTS 버전을 설치합니다.
3.  **필수 패키지 설치**:
    ```bash
    npm install
    ```
4.  **브라우저 드라이버 설치**:
    ```bash
    npx playwright install chromium
    ```

## 🚀 실행 방법

### 🍎 MacOS (맥북)
- 폴더 내의 **`Crawl.command`** 파일을 더블 클릭하면 모든 과정이 자동으로 시작됩니다.
  - 브라우저 실행 -> 서버 구동 -> 대시보드 오픈

### 💻 Windows 및 기타 환경
1.  **브라우저 실행**: 터미널에서 `sh run_browser.sh` 실행 (9222 포트 디버깅 모드).
2.  **서버 실행**: 새로운 터미널 창을 열어 아래 명령어 입력.
    ```bash
    npm start
    ```
3.  **대시보드 접속**: 웹 브라우저 주소창에 `http://localhost:3000` 입력.

## 📂 주요 파일 구조
- `unified_crawler.js`: 크롤링 로직 및 Express 서버 엔드포인트 핵심 파일.
- `public/`: 대시보드 웹 UI 소스 코드 (HTML, CSS, JS).
- `Crawl.command`: 맥 전용 원클릭 통합 실행 스크립트.
- `run_browser.sh`: 스텔스 모드 유지를 위한 디버그 크롬 실행 스크립트.
- `result.json`: 수집된 결과 데이터 저장소 (Git 관리 제외).
- `package.json`: 프로젝트 의존성 및 설정 정보.

## ⚠️ 주의사항
- **Access Denied**: 쿠팡의 강력한 차단 정책으로 인해 수집 중 차단될 수 있습니다. 이 경우 팝업으로 뜨는 크롬 창에서 캡차를 직접 풀어주시거나 IP를 변경해야 합니다.
- **Headless 모드 권장 안 함**: 차단을 피하기 위해 실제 브라우저 창이 뜨는 모드로 기본 설정되어 있습니다.
